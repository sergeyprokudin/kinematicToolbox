savestats

(beat/savestats.m)

Function Synopsis

savestats (data)

Help text

 SAVESTATS  Save data from user to a tab-delimited file.  
   If files exists, append to file.  SAVESTATS (DATA) asks the 
   user to specify a file, and saves the information in DATA 
   to that file, in ASCII.  The file is created if 
   it does not exist, or appended to if it is an existing file.
