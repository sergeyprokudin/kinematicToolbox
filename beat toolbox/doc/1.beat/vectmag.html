vectmag

(beat/vectmag.m)

Function Synopsis

out = vectmag (x)

Help text

 VECTMAG  A function that will compute the magnitude of a vector x.
  x is assumed to be a row vector.  If x is a matrix, each ROW
  of x is assumed to be a vector.
